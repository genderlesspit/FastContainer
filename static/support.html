Self-Service Troubleshooting

üîß Common Issues & Solutions

Page won't load or loads slowly

Try these steps:

  1. Refresh the page (Ctrl+F5 or Cmd+Shift+R)
  2. Clear your browser cache and cookies
  3. Try a different browser or incognito/private mode
  4. Check your internet connection
  5. Disable browser extensions temporarily

Still having issues? Report it using the form above.

Login problems or authentication errors

Try these steps:

  1. Verify your username and password
  2. Clear browser cookies for this site
  3. Try logging in from a different browser
  4. Check if Caps Lock is enabled
  5. Reset your password if needed

Still can't log in? Contact IT support at it@phazebreak.com

Forms not submitting or errors when saving

Try these steps:

  1. Check that all required fields are filled out
  2. Verify file uploads are under size limits
  3. Try submitting without attachments first
  4. Disable ad blockers or script blockers
  5. Try using a different browser
Database search not returning results

Try these steps:

  1. Check your search terms for typos
  2. Try broader search terms
  3. Clear all filters and try again
  4. Verify you have permission to access the data
  5. Try the "All Fields" search option
Downloads failing or corrupted files

Try these steps:

  1. Right-click and "Save Link As" instead of direct clicking
  2. Try downloading from a different browser
  3. Temporarily disable antivirus software
  4. Check available disk space
  5. Try downloading smaller files first

‚ùì Frequently Asked Questions

General Questions

How do I access the internal database?

The database search is available to all employees through the main navigation menu. If you're having trouble accessing it, ensure you're logged in with your company credentials and contact IT if the issue persists.

Why can't I download certain files?

File access depends on your user permissions and department. Some technical documents are restricted to engineering and sales teams. Contact your manager if you need access to specific resources.

How often is the news feed updated?

The renewable energy RSS feeds are updated every 15 minutes, while LinkedIn feeds refresh every 30 minutes. Manual refresh buttons are available if you need immediate updates.

Can I use the website on my mobile device?

Yes! The website is fully responsive and optimized for mobile devices. Some advanced features work best on tablets or desktops, but all core functionality is available on mobile.

Technical Questions

Which browsers are supported?

We support Chrome 100+, Firefox 100+, Safari 15+, and Edge 100+. Internet Explorer has limited support. For the best experience, please use the latest version of your preferred browser.

Why am I getting certificate warnings?

This typically happens when accessing the site from an untrusted network. Ensure you're using the correct URL (https://phazebreak.com) and contact IT if the warnings persist from the office network.

How do I clear my browser cache?

Chrome: Ctrl+Shift+Delete, select "All time", check "Cached images and files"
Firefox: Ctrl+Shift+Delete, select "Everything", check "Cache"
Safari: Safari menu > Clear History, select "All History"

What should I do if the website is completely down?

First, check if it's just your connection by trying other websites. If the site is truly down, email it@phazebreak.com or call the IT help desk at ext. 911. We monitor uptime 24/7.

Account & Access Questions

I forgot my password. How do I reset it?

Use the "Forgot Password" link on the login page, or contact IT support. For security reasons, only IT can reset passwords for internal accounts.

Why do I see "Access Denied" on some pages?

Page access is controlled by department and role. Sales staff can access customer data, engineering can access technical documents, etc. Contact your manager if you need additional permissions.

Can external partners access our website?

We have a separate portal for approved partners and vendors. Internal company pages are restricted to employees only. Contact business development for partner access requests.
